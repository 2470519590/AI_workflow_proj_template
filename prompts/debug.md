# debug.md

请按本项目 AI 工作流调试问题。

要求：

- 先读取 `AGENTS.md`、`AI_README.md`、`docs/INDEX.md`、`docs/CODING_RULES.md`、`docs/DEBUG_GUIDE.md`、`docs/ERROR_CATALOG.md`。
- 先定位疑似归属模块和风险区域，再提出修改方案。
- 区分观察事实、推测和待验证假设。
- 不要把日志全文复制进文档，只摘录关键片段。
- 未确认根因前，不要随机修改代码。
- 小 debug 过程不要单独写任务流水账。

请输出：

- 现象复述
- 已读上下文
- 关键观察
- 疑似归属
- 风险区域
- 下一步最小检查
- 是否需要改代码
